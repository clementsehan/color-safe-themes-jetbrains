# ColorSafe Themes Changelog

## [Unreleased]

## [0.0.4] - 2026-04-24

### Fixed

- Plugin logo now displays correctly in the IDE plugin manager — replaced PNG with an SVG icon, which is the only format rendered by the IDE

## [0.0.3] - 2026-04-24

### Fixed

- Plugin icon now displays correctly in the IDE plugin manager and JetBrains Marketplace — resized to the required 40×40 px, with an 80×80 px HiDPI variant added

## [0.0.2] - 2026-04-23

### Changed

- Improved Marketplace plugin description: clearer feature breakdown, added IDE compatibility list, and SEO keywords for better discoverability

## [0.0.1]

### Added

- **Pro-Deutan Dark** theme — high-contrast dark theme engineered for Deuteranopia and Protanopia
- Blue-Amber syntax axis replaces the red-green distinctions that are invisible to colorblind developers
- WCAG AA contrast ratios throughout the editor, UI panels, and menus
- Triple-encoded errors: distinct color + squiggly underline shape + bold font weight
- Squiggly underline differentiation between errors and warnings
- Bold keywords for instant visual hierarchy
- Fixed contrast for selected text in IDE menus and modals
